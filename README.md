# Дипломная работа: API сервис заказа товаров для розничных сетей

## Описание проекта
Данный проект представляет собой backend-часть информационной системы для автоматизации процесса закупок в розничной сети.  

Сервис позволяет:
- магазинам просматривать каталоги товаров
- управлять корзиной
- оформлять заказы  
- поставщикам — обновлять прайс-листы через YAML-файлы

---

## Стек технологий
- Python 3.13 / Django 6.0.2
- **Celery & Redis** (асинхронная обработка задач)
- **DRF Spectacular** (OpenAPI 3.0 / Swagger UI)
- **Sentry** (мониторинг ошибок)
- SQLite (база данных включена в репозиторий для удобства проверки)

## Ключевая функциональность

### 1. Асинхронные уведомления (Celery)
- Все тяжелые операции (отправка писем при регистрации, подтверждении почты или создании заказа) вынесены в фоновые задачи Celery.
- Это обеспечивает мгновенный отклик API для пользователя.

### 2. Мониторинг и безопасность
- **Sentry**: Интегрирована система отслеживания ошибок.
- **Throttling**: Настроены ограничения частоты запросов для защиты от перегрузок и спама.

## Как запустить проект

### 1. **Клонировать репозиторий**:
```bash
git clone https://github.com/Daria-Bog/netology-python-diplom-api.git
```
### 2. **Установить зависимости**:
```bash
pip install -r requirements.txt
```
### 3. **Запустить сервер**:
```bash
python manage.py runserver
```
### 4. Запустить Celery (в отдельном терминале):
```bash
celery -A netology_pd_diplom worker -l info -P eventlet
```
